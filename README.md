# tokokita_Mufthie Alie - H1D023042


Aplikasi ini menampilkan daftar produk dan menyediakan fungsi untuk melihat detail produk, menambah data baru, mengubah data, dan menghapus produk. Pada ProdukPage, semua produk ditampilkan dalam bentuk list, dan ketika ditekan, pengguna diarahkan ke ProdukDetail. Di halaman detail, terdapat tombol Edit untuk membuka form pengubahan data, dan tombol Delete yang menampilkan konfirmasi sebelum menghapus data melalui fungsi ProdukBloc.deleteProduk().

Halaman ProdukForm berfungsi sebagai form input untuk menambah atau mengubah produk. Jika form menerima data produk, maka field otomatis terisi dan tombol berubah menjadi mode edit. Validasi digunakan untuk memastikan semua input sudah terisi dengan benar sebelum diproses.

Selain fitur produk, aplikasi ini juga memiliki halaman login dan registrasi. Halaman login mengecek input email dan password, sedangkan halaman registrasi memiliki validasi tambahan seperti email valid, minimal karakter, dan kecocokan password. Navigasi antarhalaman menggunakan Navigator.push() untuk berpindah ke halaman berikutnya berdasarkan aksi pengguna.
